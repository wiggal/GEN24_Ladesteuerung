; Genauere Erklärungen siehe config.ini.info
[Ladeberechnung]
; HTTP und MODBUS
; Wenn die Variable "FesteLadeleistung" größer/gleich "0" Watt ist, wird der Wert geschrieben.
; FesteLadeleistung = -1, Ladesteuerung steht auf AUTO.
FesteLadeleistung
; Bis wann (Stunde) soll die Batterie voll sein, ganze Zahl > 0 ist Stunde
; Zahlen <= 0 wird Differenz zum Sonnenuntergang (= Prognosewert < 2% PV_Leistung_Watt)
BattVollUm
; Faktor um Batteriekapazität für später aufzusparen.
; Der aus der Prognose errechnete Batterieladewert wird mit dem BatSparFaktor multipliziert.
BatSparFaktor
; Größter Batterieladewert, der im WR gesetzt werden soll, aber höchstens der maximale mögliche Ladewert!!!
MaxLadung
; die Einspeisegrenze sollte etwas unterhalb der wirklichen Grenze liegen (1-2%).
Einspeisegrenze
; AC Kapazität des Wechselrichters um den Überschuss in die Batterie zu laden (etwas darunter bleiben 1-2%)
WR_Kapazitaet
; PV-Leistung reellen Spitzenwert
PV_Leistung_Watt
; Grundlast wird verwendet um den Wert "aktuellerUeberschuss" zu berechnen
Grundlast
; Grundlast je nach Wochentag wird gesetzt, wenn Grundlast == 0
; Wochentage = Mo, Die, Mi, Do, Fr, Sa, So
Grundlast_WoT
; Schaltverzögerung um Schreibzugriffe auf WR zu minimieren
WRSchreibGrenze_nachOben
WRSchreibGrenze_nachUnten
; Liegt der Batteriestatus unter MindBattLad in % wird voll geladen
MindBattLad
; Wenn Ladung ausgeschaltet den Wert LadungAus in Watt setzen (z.B. 0 oder 10 )
LadungAus
; Wenn größter Prognosewert je Stunde ist kleiner als GrenzwertGroestePrognose volle Ladung (= MaxLadung)
GrenzwertGroestePrognose
; Akkuschonung (0=aus, 0.1 bis 1.0 ein) dadurch wird der Akku zwischen 80 und 95% mit 0,2C bzw. 0,1C geladen
; Ab 95% wir mit 0,1C mal Akkuschonung (z.B.: 0.1x0.5 = 0.05C) geladen
; und BattVollUm wird um eine Stunde vor verlegt, da am Schluss nicht mehr so stark geladen wird.
; ACHTUNG: Eine Überschreitung der Einspeisegrenze bzw. AC-Leistung des WR setzt die Akkuschonung aus!
Akkuschonung
; hier müssen die Namen der zusätzlichen Konfigurationen definiert werden.
; mit Zusatz_Ladebloecke = aus können die zusätzlichen Konfigurationen abgeschaltet werden
; Zusatz_Ladebloecke = aus
Zusatz_Ladebloecke

Monate
