Beschreibung Config-Parameter
TAB--> config

In der Überschrift wird die installierte Version der Ladesteuerung angezeigt.
Wenn das Quellverzeichnis ein Git-Repository ist, also mit `git clone https://github.com/wiggal/GEN24_Ladesteuerung.git .` erzeugt wurde (siehe install im WIKI), wird nach der Überschrift ein Updatebutton angezeigt. Wenn eine neuere Version auf Github vorhanden ist, kann über diesen Button ein Update gemacht werden.

In dem Dropdown können alle vorhandenen _priv.ini-Dateien ausgewählt und mit dem Button angezeigt werden. Anschließend kann die ausgewählte _priv.ini-Datei editiert oder wenn weine original ini-Datei vorhanden ist mit dieser verglichen und auf deren Stand upgedatet werden.
