🌍 Produktionswerte von fremden Erzeugern:

Hiermit können Produktionsdaten von fremden Erzeugern (z.B. Balkonkraftwerke) mit eigenem Skript (ADDONS/Fremd_API_priv.py) in GEN24_Ladesteuerung integriert werden.
Die Beschreibung ADDONS/Fremd_API_priv.readme erklärt den Weg.

📉 Grafana von @Manniene:

Die Integration der Daten aus GEN24_Ladesteuerung wurde von @Manniene ausgearbeitet.
Die Beschreibung (Grafana_Installation_readme.pdf) und Beispieldashboards liegen im Verzeichnis GRAFANA.

Anmerkungen und Fragen dazu bitte in folgender Diskussion.

🌦️ Wetterdaten von Homeassitant holen von @roethigj:

Die Beschreibung (Solcast_WeatherData_from_HA_file.readme) und das Pythonskript (Solcast_WeatherData_from_HA_file.py) zum Ausführen als Cronjob liegen im Verzeichnis ADDONS.
Damit können die Wetterdaten von Homeassistant geholt werden, um nicht unnötige Requests zu verbrauchen.

🏠 Home Assistant Add-on erstellt von @roethigj.
